# 实战踩坑日志（Field Notes）

> **机制**：每做完一份真实 deck（尤其翻过车、被换眼 QA 揪出问题、或客户反馈后改的），在此**追加一条带日期的记录**。一条 = 症状 + 根因 + 修法 + 适用身份。
> 目的：让 skill 像老师傅一样**累积里程**——同样的坑只踩一次。新坑沉淀在这里，严重的上升为 `design-identities.md` 配方或 `checklist` / `visual-qa.md` 的硬规则。
>
> 记录格式：
> ```
> ## YYYY-MM-DD · 身份X · 一句话标题
> - 症状：渲染后看到的具体问题
> - 根因：为什么会这样
> - 修法：怎么改的（可复用的规则）
> - 已上升为规则？：是/否（上升到哪个文件）
> ```

---

## 2026-06-18 · 元规则 · 同一个 agent 看不出自己挤了

- 症状：自检清单全过、自我感觉良好，但渲染成图后到处是拥挤/塌底/溢出。
- 根因：写版面的 agent 有"预期盲区"——它知道每个元素**应该**在哪，于是脑补成对的。
- 修法：交付前**强制**渲染成图 + 派一个**全新子 agent**只看图挑刺，修完重验。
- 已上升为规则？：是 → `visual-qa.md` + SKILL.md 核心原则 #12。

## 2026-06-18 · 身份 B Keynote · 白场页大字"隐身"

- 症状：深色身份里插了一张亮底（paper）页，大标题仍是白字 → 几乎不可见。
- 根因：深色身份默认文字色是高亮白，切到亮底页没翻转文字色。
- 修法：亮底页（.paper / 白场）强制文字改深色；状态色同步调暗版。
- 已上升为规则？：否（属模板内已修；新建深色身份时注意此点）。

## 2026-06-18 · 身份 B/C · 强调色失控

- 症状：半句标题 + 整组数字全染成强调色，"单一锚点色"原则被破坏，画面花。
- 根因：强调色当成了"重点色"到处用。
- 修法：每页强调点**只留 1 个**（最该看的那一项），其余回到 ink/中性。
- 已上升为规则？：是 → design-identities 各身份"单一强调"已有；checklist 红色≤3 次。

## 2026-06-18 · 全身份 · 留白"塌底"而非"分布"

- 症状：内容堆在上半页，下半页大片空白，显得"没做完"（不是呼吸，是失衡）。
- 根因：内容区在 `1fr` 行里 `align-content:start` 顶到上方；内容少时底部塌空。
- 修法：内容偏少的页给正文区加 `.center-y` 纵向居中，留白分到上下。
- 已上升为规则？：是 → 模板 `.center-y` 工具 + `visual-qa.md` 挑刺项 + layouts 提醒。

## 2026-06-18 · 身份 B · 数字与单位相撞

- 症状：超大数字（"0.18"）与单位（"秒"）挨太近/基线不齐，糊成一团。
- 根因：单位直接贴在大数字后，没给间距、没按 cap-height 对齐。
- 修法：单位加间距、抬到接近数字 cap-height 处，作为独立单位读。
- 已上升为规则？：否（个案手法，记此备查）。

## 2026-06-18 · 身份 C Editorial · 页脚被底边截断

- 症状：底部三栏元数据/刊尾被画布下边缘切掉。
- 根因：内容贴到 100% 高度，没留底部呼吸边距。
- 修法：内容纵向收在 ~90% 以内，页脚留足底边距（呼吸 #11）。
- 已上升为规则？：是 → 呼吸量化（内容收在纵向 10–90%）。

## 2026-06-18 · 身份 D Dark-tech · 双层嵌套卡塌成单层描边

- 症状：本该有"玻璃板嵌在金属托盘"的双层质感，渲染出来只是一个描边框。
- 根因：内核没有独立填充色 + 缺 inset 顶部高光，外壳内核没拉开层次。
- 修法：内核填更深底色 + 加 `inset 0 1px ... rgba(255,255,255,.15)` 顶缘高光，圆角同心（外大内小）。
- 已上升为规则？：是 → design-identities D 节 Doppelrand 配方。

## 2026-06-18 · 字体 · 配方写了商用字体 / 本机没装就回退

- 症状：spec 写 PP Editorial New（商用）；本机未装免费字体时 headless 截图回退系统字体，不够锐利。
- 根因：①引用了不可免费分发的字体；②未本机安装、离线渲染拿不到 web 字体。
- 修法：①全改免费 OFL 字体（Fraunces/Playfair/Geist/Plus Jakarta/JetBrains Mono），模板 CDN 加载；②要离线 crisp 就装进 `~/Library/Fonts/`。
- 已上升为规则？：是 → design-identities「字体获取」节。
